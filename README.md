# Usage
<!-- usage -->
```sh-session
$ npm install -g avweather-cli
$ avwx COMMAND
running command...
$ avwx (-v|--version|version)
avweather-cli/0.0.1 darwin-x64 node-v10.11.0
$ avwx --help [COMMAND]
USAGE
  $ avwx COMMAND
...
```
